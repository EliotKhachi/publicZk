Linker Directives

Linker directives tell the Linker Linker where the available memory is and how they should be used. They reflect exactly the memory resources and memory map of the target microcontroller Memory Map of a Microcontroller.

Kinds of Directives

  • LIBPATH
  • FILES
  • CODEPAGE
  • ACCESS BANK
  • DATABANK
  • SECTION
  • STACK SIZE
