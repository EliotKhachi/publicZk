# Linker Directive: STACK SIZE 

The *STACK SIZE* directive specifies the software stack location and size.   

## Syntax
```c
STACK SIZE=0x100 RAM=gpr2
```  

Here a stack size of 100 is specified, located in RAM Bank 2.
