# #include Directive

The `#include` directive is used in C/C++ to tell the *preprocessor* [\[Preprocessor\]](../202202151838) to include the contents of a specified file at the point where the directive appears.  

## Syntax
`#include "` *path-spec* `"`  
`#include <` *path-spec* `>`
