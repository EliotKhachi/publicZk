OpenSSL Adding a Certificate Authority

Steps

  1. First create a root certificate --> Generate a Self-Signed SSL Certificate, and then add the certificate to your browser's list of Certificate Authorities (CAs) --> 202206112007 - Configuring Certificate Authorities

  2. Next, you must generate a certificate signing request (CSR), and use that to create your server certificate.
