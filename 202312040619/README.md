# Setting up the Raspberry Pi Pico

1. Get the SDK:  
```
git clone https://github.com/raspberrypi/pico-sdk.git --branch master
cd pico-sdk
git submodule update --init
```

2. Get the examples  
```
git clone https://github.com/raspberrypi/pico-examples.git --branch master
```
