Linker Script

A linker script is a text file written in the ld command language [Linker Command Language: ld] made up of a series of directives [Linker Directives].

Example Linker Script for STM32F030x8

MEMORY
{
    FLASH(rx)   : ORIGIN = 0x08000000, LENGTH=64K
    RAM(xrw)    : ORIGIN = 0x20000000, LENGTH=8K
}

_estack = ORIGIN(RAM) + LENGTH(RAM);

SECTIONS
{
    .isr_vector:
    {
        . = ALIGN(4);
        KEEP(*(.isr_vector))
        . = ALIGN(4);
    }
    
    .text :
    {
        . = ALIGN(4);
        *(.text)
        *(.text*)
    }
